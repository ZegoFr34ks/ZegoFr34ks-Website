Description:

This a game I created on Unreal Engine using Blueprint, there are no textures yet, because some mechanics need yet to be implimented, however, this project is on hold right now. The game is functional, and took me about 1 week to make. It's still a work in progress. It's based on Five Nights at Freddy's gameplay with some unique features. In this project you have a working camera system, the ability to close the door through which can come the ennemy, flashlights to scare them off and see and a generator that you'll need to recharge every once and a while to keep the power up. There is also a vent from which the ennemy can come from, you'll have to listen to the distinct audio cues that will inform the player how near he is (as far as for now there is no mechanic to counter him yet). There are also no actual ennemies yet that are attacking you, so you can't lose at the moment.
